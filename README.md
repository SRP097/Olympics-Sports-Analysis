# Olympics Sports Analysis

## Project Overview
The Olympics Sports Analysis project is a comprehensive data analytics and business intelligence project focused on analysing the historical Olympic Games dataset. The project explores trends, participation patterns, medal distributions, sports evolution, athlete demographics, country performance, and Olympic hosting history using SQL, Excel, and Power BI.

The objective of this project is to transform raw Olympic historical data into meaningful insights through exploratory data analysis, KPI evaluation, interactive dashboards, and data visualization techniques.

---

## Problem Statement
The project aims to analyse Olympic Games history across multiple dimensions such as:

- Games evolution over time
- Sports and event trends
- Athlete demographics
- Country-wise performance
- Medal analysis
- Participation trends
- Gender diversity
- Regional dominance
- Historical Olympic patterns
- Interactive dashboard reporting

# Key Insights

- USA consistently dominates Olympic medal tally.
- Athletics contains the highest number of Olympic events.
- Rugby Sevens emerged as a newly introduced sport.
- Several sports lost popularity over decades.
- Male participation historically exceeded female participation.
- Countries like USA, UK, Germany, and Russia consistently perform strongly across       multiple Olympic editions.
- Athlete height and weight show increasing trends over time.
